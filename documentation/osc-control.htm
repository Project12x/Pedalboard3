OSC Control

Control Pedalboard3 remotely using Open Sound Control (OSC) from tablets, phones, or other computers.

Enabling OSC

  1. Go to Options > Misc Settings
  2. Enable OSC input
  3. Note the port number (default: 8000)
  4. Your IP address is shown in Help > About

Compatible Apps

  • TouchOSC (iOS, Android) - Custom layouts
  • Open Stage Control (Desktop) - Free, open source
  • OSC Controller (iOS) - Simple controls

OSC Address Format

Parameters are addressed using this format:

/pedalboard/plugin/[plugin-name]/param/[param-index]

Mapping OSC

  1. Right-click on any plugin parameter
  2. Select OSC Learn
  3. Send an OSC message from your controller
  4. The mapping is created automatically

Application Mappings

Go to Options > Application Mappings to map OSC to:

  • Patch switching
  • Transport controls
  • Master volume
  • Any plugin parameter

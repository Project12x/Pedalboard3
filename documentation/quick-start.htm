Quick Start Guide

1. Configure Audio

  1. Go to Options > Audio Settings
  2. Select your audio interface for Input and Output
  3. Choose a sample rate (44100 or 48000 Hz recommended)
  4. Set buffer size (lower = less latency, higher = more stability)

2. Add Plugins

  1. Go to Options > Plugin Scanning to scan for VST3 plugins
  2. Right-click on the canvas to add plugins
  3. Connect plugins by dragging from output to input nodes

3. Route Audio

  1. Connect Audio Input to your first effect
  2. Chain effects together
  3. Connect the last effect to Audio Output
Tip: Use Ctrl+S to save your patch, and Ctrl+O to open saved patches.

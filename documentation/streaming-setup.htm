Streaming & Recording Setup

How to capture Pedalboard3 audio in OBS, Discord, and other applications.

Recommended: OBS Application Audio Capture

Recommended

OBS Studio (Windows 10/11)

OBS 28+ can capture audio directly from Pedalboard3 without any additional software.

Setup Steps:

  1. Open OBS Studio (version 28 or newer)
  2. In the Sources panel, click the + button
  3. Select Application Audio Capture
  4. Name it (e.g., "Pedalboard Audio") and click OK
  5. In the properties window, find Pedalboard3 in the application list
  6. Click OK to add the source
Note: Make sure Pedalboard3 is running and producing audio before adding the source. OBS can only detect applications that are actively using audio.

Advantages:

  • Zero additional software required
  • Captures only Pedalboard3 audio (not system sounds)
  • No latency added
  • Works immediately

Alternative Methods

Alternative

VB-Cable (Virtual Audio Cable)

A free virtual audio driver that creates a virtual audio device for routing.

Setup Steps:

  1. Download and install VB-Cable (free)
  2. Restart your computer
  3. In Pedalboard3, go to Options > Audio Settings
  4. Set the Output to CABLE Input (VB-Audio Virtual Cable)
  5. In OBS, add an Audio Input Capture source
  6. Select CABLE Output (VB-Audio Virtual Cable)
Tip: You won't hear audio through your speakers when outputting to VB-Cable. Use VoiceMeeter (also free from VB-Audio) if you need to hear audio while streaming.
Alternative

Discord / Zoom / Video Calls

To share Pedalboard3 audio in voice/video applications:

  1. Install VB-Cable (see above)
  2. Set Pedalboard3 output to CABLE Input
  3. In Discord/Zoom, set your Input Device to CABLE Output

This routes your processed audio as if it were a microphone input.

Troubleshooting

No audio in OBS?

  • Check that Pedalboard3 is producing audio (meters should be moving)
  • Make sure the audio source isn't muted in OBS
  • Try restarting OBS after opening Pedalboard3

Audio is delayed/out of sync?

  • In OBS, right-click the audio source and select Advanced Audio Properties
  • Adjust the Sync Offset to match your video

VB-Cable not appearing?

  • Restart your computer after installing
  • Check Windows Sound Settings to ensure VB-Cable is enabled
  • Run the VB-Cable installer as Administrator

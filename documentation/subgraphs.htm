Subgraphs

Subgraphs let you organize complex signal chains into reusable, collapsible groups.

Creating a Subgraph

  1. Right-click on the canvas
  2. Select Built-in > Subgraph
  3. Double-click the subgraph node to open it
  4. Add plugins inside the subgraph
  5. Connect Input and Output nodes to define the signal flow

Use Cases

  • Effect chains: Group your go-to pedal combinations
  • Parallel processing: Create a wet/dry mix inside a subgraph
  • Organization: Keep complex patches clean and readable
  • Reuse: Copy subgraphs between patches

Navigation

  • Double-click a subgraph to enter it
  • Use the breadcrumb bar at the top to navigate back
  • Or click outside the subgraph bounds
Tip: Name your subgraphs descriptively (right-click > Rename) so you remember what's inside.

I/O Configuration

Subgraphs can have multiple audio and MIDI inputs/outputs:

  • Add Input/Output nodes inside the subgraph
  • These appear as connection points on the subgraph node
  • Useful for stereo processing, sidechain, or MIDI routing

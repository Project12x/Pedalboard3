MIDI Mapping

Control plugin parameters with MIDI controllers, foot pedals, and other MIDI devices.

Quick MIDI Learn

  1. Right-click on any plugin parameter slider
  2. Select MIDI Learn
  3. Move your MIDI controller (knob, fader, or pedal)
  4. The parameter is now mapped!

MIDI Mapping Manager

For advanced mapping, go to Options > Application Mappings.

Here you can:

  • View all current MIDI mappings
  • Edit mapping ranges and curves
  • Delete mappings
  • Map MIDI to application functions (patch switching, transport, etc.)

Expression Pedals

Expression pedals send continuous controller (CC) messages, typically CC 1, 7, or 11.

  1. Connect your expression pedal to your MIDI interface
  2. Use MIDI Learn on the parameter you want to control
  3. Move the pedal to complete the mapping
Tip: You can map multiple parameters to the same MIDI controller for linked control.

Foot Switches

MIDI foot switches can be mapped to:

  • Plugin bypass (toggle on/off)
  • Patch switching (next/previous)
  • Transport controls (play/pause)
  • Tap tempo
